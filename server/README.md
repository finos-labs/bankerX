# Server
This module runs the logic against a Netty-based server, and can be run locally and/or outside of AWS.

To run this locally, use:
```sh
./mill server.run
```

<details>

<summary>Building for Windows users</summary>

```sh
mill.bat server.run
```

or using Powershell

```sh
.\mill.ps1 server.run
```

</details>
